Tutorial membuat game simple menggunakan Pygame

Python Game


Gimana sih caranya bikin game simple dengan cara menggunakan library pygame?

Kalian tidak perlu menginstall hal yang ribet cukup duduk dan simak saja. Karena pada artikel ini kita akan belajar cara membuat game di Linux/Windows dengan PyGame. Siap? Mari kita mulai..

Persiapan Awal yang Harus dilakukan


Sebelum memulai, saya akan menjelaskan dulu persiapan yang harus dilakukan. Pada tutorial ini, kita akan menggunakan modul PyGame dengan Python 3. Jadi ada dua hal utama yang harus disiapkan:

  • Python 3
  • PyGame

PyGame adalah modul Python yang berisi fungsi dan class yang kita butuhkan untuk membuat game.

Pengetahuan yang dibutuhkan


Setidaknya untuk dapat mengikuti tutorial ini, kamu harus paham terlebih dahulu dasar-dasar dari pemrograman Python.

Apabila belum paham, silahkan ikuti Tutorial Pemrograman Python dasar. Selain itu pengetahuan tentang grafika komputer dan matematika juga akan sangat membantu.
